## Sobre

Site de portfólio pessoal construído com Next.js 16 (App Router) e TypeScript, com foco em performance e identidade visual consistente.

Disponível em **[nycocado.dev](https://nycocado.dev)**.

## Como executar

```bash
npm install
npm run dev    # dev server em http://localhost:3000
npm run build  # build de produção
npm run lint   # ESLint
```
